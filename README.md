### What I learned

- Use flexbox to center /align elements on the page
  - `content-justify`
  - `align-items`
- Align text within an element: `text-align`
- Ensure `<body>` takes up 100% of the viewport:

  ```css
  body {
    margin: 0;
    height: 100vh;
  }
  ```
